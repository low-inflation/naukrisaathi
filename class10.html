📚 Key Exams After Class 10

After you pass Class 10 (Matriculation), there are many government exams you can try for. These exams help you get good jobs in the central government or your state.

🏛️ Central Level Exams (All India)

  • 1. SSC MTS & Havaldar
    ➡️ Office helper jobs & Havaldar posts in central govt departments.
    ✅ Class 10 pass
    🌍 All India
    💰 ₹23,000–₹25,000/month
  • 2. SSC GD Constable
    ➡️ Join BSF, CRPF, CISF, ITBP, SSB, or SSF.
    ✅ Class 10 pass
    💪 Physical test needed
    💰 ₹30,000–₹32,000/month
  • 3. RRB Group D
    ➡️ Railway jobs like helper, track maintainer, porter.
    ✅ Class 10 pass
    🚆 Railway lovers
    💰 ₹22,000–₹25,000/month
  • 4. Indian Post GDS
    ➡️ Post office jobs in villages/towns.
    ✅ Class 10 pass with good marks in Maths & English
    📬 No exam
    💰 ₹10,000–₹14,500/month
  • 5. Army Agniveer
    ➡️ Soldier GD & Tradesman roles.
    ✅ Class 10 pass
    📋 Physical + written test
    💰 ₹30,000–₹40,000/month
  • 6. Navy Agniveer (MR)
    ➡️ Matric Recruit posts like cook, steward.
    ✅ Class 10 pass
    🚢 Physical + written test
    💰 ₹30,000–₹40,000/month
  • 7. Air Force Agniveer Vayu
    ➡️ Trades/support roles.
    ✅ Class 10 pass
    ✈️ Physical + written test
    💰 ₹30,000–₹40,000/month

🌾 Assam State Level Exams

  • 8. Assam Direct Recruitment – Grade IV
    ➡️ Peon, Chowkidar, Driver, etc.
    ✅ Class 8/10 pass
    🏢 No special skill
    💰 ₹17,000–₹20,000/month
  • 9. Assam Police Grade IV
    ➡️ Peon, Chowkidar, Driver.
    ✅ Class 10 pass
    🚓 Physical + written test
    💰 ₹18,000–₹21,000/month
  • 10. Assam Forest Guard / AFPF Constable
    ➡️ Protect forests/wildlife.
    ✅ Class 10 pass
    🌲 Physical + written test
    💰 ₹22,000–₹25,000/month
  • 11. Excise Constable
    ➡️ Excise Dept jobs.
    ✅ Class 10 pass
    🔍 Written + Physical test
    💰 ₹22,000–₹25,000/month
  • 12. Home Guard Volunteer
    ➡️ Assist police.
    ✅ Class 10 pass
    🎽 Physical fitness test
    💰 ₹9,000–₹11,000/month
