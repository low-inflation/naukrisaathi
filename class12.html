📘 Government Jobs after Class 12

Your guide for HS-level (Class 12) government job options — links, tips, important dates

← Back to Home
Note: This page lists only jobs where Class 12 is the minimum eligibility. If you have Class 12, you are also eligible for many Class 10-level jobs (see Class 10 page).

🏛️ Central Level Exams (All India)

  • 1. SSC CHSL (Combined Higher Secondary Level) ➡️ LDC, DEO, Postal Assistant, Court Clerk. ✅ Class 12 pass • 🌍 All India • 💰 ₹25,000–₹40,000/month
  • 2. SSC Stenographer (Grade C & D) ➡️ Stenography & clerical roles. ✅ Class 12 pass + stenography skill • 🌍 All India • 💰 ₹30,000–₹45,000/month
  • 3. NDA (National Defence Academy) / NA (Naval Academy) ➡️ Officer entry after training. ✅ Class 12 pass (PCM for Navy/Air Force) • 🌍 All India • 💰 Training stipend + allowances
  • 4. Indian Army Agniveer (Technical & Clerk/SKT) ➡️ Agniveer Technical (Class 12 Science required) and Clerk/SKT (Class 12 any stream). ✅ Class 12 pass • 🌍 All India • 💰 Allowances + post-service opportunities
  • 5. Indian Air Force Agniveer Vayu ➡️ Technical & non-technical trades for HS pass candidates. ✅ Class 12 pass (stream as specified) • 🌍 All India
  • 6. Indian Navy Agniveer (SSR) ➡️ Technical sailor trades. ✅ Class 12 pass (science required for some trades) • 🌍 All India
  • 7. Indian Coast Guard – Navik (General Duty) ➡️ Sailor roles, maritime security. ✅ Class 12 pass (specified streams for some posts) • 🌍 All India
  • 8. RRB NTPC (Level 2/3 posts e.g., Clerk, Typist, Ticket) ➡️ Non-technical popular categories requiring HS for some posts. ✅ Class 12 pass for select positions • 🌍 All India
  • 9. IB Security Assistant / UDC (where HS required) ➡️ Intelligence Bureau clerical/support roles. ✅ Class 12 (for some posts) • 🌍 All India

🌾 Assam State Level Exams

  • 1. Assam Secretariat – Junior Administrative Assistant (JAA) ➡️ Clerical & admin assistant work in state departments. ✅ Class 12 pass • 🏢 Assam-specific
  • 2. Assam Irrigation Department – Junior Assistant ➡️ Office clerical roles in irrigation & water resources. ✅ Class 12 pass • 🏢 Assam-specific
  • 3. Assam Forest – Forester / Forester Grade I ➡️ Field & protection duties; some posts require Class 12 (science). ✅ Class 12 (science for some posts) • 🌲 Assam
  • 4. Assam Education Dept. – HS Level Teacher (eligibility varies) ➡️ Teaching roles at higher secondary level (requires HS + professional qualification as specified). ✅ Class 12 + required diploma/BT/CERT • 🏫 Assam
  • 5. APDCL / AEGCL – Junior Assistant / HS-level technical & clerical posts ➡️ Office & technical support posts in power utilities. ✅ Class 12 pass • 🏭 Assam

💡 Quick Tips for Class 12 Candidates

  • Start fitness & basic medical checks if applying to armed forces (PFT/PMT require time).
  • Keep scanned documents, Class 12 mark sheet, and ID ready before application windows open.
  • Practice mock tests for reasoning, maths, and English for CHSL/NTPC.
  • Follow official websites and subscribe to alerts — do not rely on unofficial posts only.
